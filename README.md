# Exploratory Data Analysis (EDA) — World Population (1970 - 2022)

## Project Overview
This repository contains an end-to-end Exploratory Data Analysis (EDA) of global population trends spanning over five decades. The objective of this project was to clean, transform, reshape, and analyze demographic data to uncover macroeconomic insights regarding global growth patterns, continental shifts, and extreme national outliers.

## Dataset Source
* **Dataset:** World Population Dataset (Kaggle)
* **Scope:** 234 Countries and Territories across 8 historical intervals (1970–2022).

## Key Tech Stack
* **Python** (Core Analysis)
* **Pandas & NumPy** (Data Cleaning, Feature Engineering, Data Melting/Reshaping)
* **Matplotlib & Seaborn** (Data Visualization & Advanced Plotting)

## Key Findings
1. **Global Doubling:** The human population doubled in just over 50 years, climbing from ~3.7 billion in 1970 to nearly 8 billion in 2022.
2. **Growth Trajectories:** Asia holds the majority share of the global population (~59%), while Africa represents the fastest-growing continent by relative acceleration. 
3. **Extreme Outliers:** Feature engineering revealed that the United Arab Emirates experienced the highest percentage growth since 1970, driven by rapid urbanization, while parts of Eastern Europe are facing significant population contraction.
